# My Circles of Care Website

A modern, accessible website for My Circles of Care - NDIS Provider on the Central Coast of NSW.

**Live URL:** [To be added after deployment]

## Features

- **Next.js 14** with App Router for optimal performance
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **SEO Optimized** with schema markup, meta tags, and Open Graph
- **Accessible** (WCAG 2.1 compliant)
- **Responsive** design for all devices
- **Fast** with static generation where possible

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel

## Pages

- `/` - Homepage with services overview
- `/about` - About My Circles of Care
- `/services` - All NDIS services
- `/services/[slug]` - Individual service pages
- `/contact` - Contact form and information
- `/faq` - Frequently asked questions
- `/ndis-resources` - NDIS guides and resources
- `/for-families` - Information for families
- `/locations/[slug]` - Location-specific landing pages

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/mcoc-website.git

# Navigate to project directory
cd mcoc-website

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
npm run build
npm start
```

## Deployment to Vercel

### Option 1: Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

### Option 2: GitHub Integration

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Vercel will auto-detect Next.js and configure build settings
5. Click Deploy

## Environment Variables

Create a `.env.local` file for local development:

```env
# Add any environment variables here
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

For production (Vercel dashboard):
- `NEXT_PUBLIC_SITE_URL`: Your production URL

## Customization

### Colors

Edit `tailwind.config.ts` to change brand colors:

```ts
colors: {
  primary: {
    500: '#ff5d10', // Main brand orange
  },
  secondary: {
    500: '#375278', // Dark blue
  },
}
```

### Content

Edit `src/lib/utils.ts` to update:
- Site configuration (name, phone, email, address)
- Services list
- Testimonials
- Statistics
- FAQs
